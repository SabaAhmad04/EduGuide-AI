# 🎓 EduGuide AI

AI-powered career guidance platform that helps students discover suitable career paths, identify skill gaps, and generate personalized learning roadmaps.

## 🚀 Live Demo

[Live Website](YOUR_RENDER_LINK)

## 📌 Problem Statement

Many students struggle to choose the right career path and often lack personalized guidance regarding skills, learning resources, and career planning.

EduGuide AI solves this problem by analyzing a student's profile and generating AI-powered recommendations tailored to their goals and interests.

- Identify skill gaps
- Create personalized learning roadmaps

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/SabaAhmad04/EduGuide-AI.git
```

### Install Dependencies

```bash
npm install
cd client && npm install
cd ../server && npm install
```

### Environment Variables

Create a `.env` file inside the server folder.

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run Project

```bash
npm start
```
